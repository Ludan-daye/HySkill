## 2. 三条经验规律（试点级证据）

**规律 A（映射 × 金标结构）**：单金标域（theoremqa/logicbench/medcalc）单向量胜；多金标域（champ/bigcodebench）四路融合胜。跨域平均：单路赢 R@1（0.48 vs 0.37，精排强），四路赢 R@10（0.82 vs 0.73，召回强）→ 提示两阶段架构（融合召回 + 单路精排）或按金标结构自适应挂挡。

**规律 B（粒度 × 域抽象度）**：抽象概念域（logic/champ）短想象胜（一句话点名即中，长文稀释）；具体技术域（code/theorem）长想象胜（细节承载信号）。完整技能形态**最鲁棒**（无塌方域，平均最高 0.675）；一句话成本 1/10 且在 2/5 域夺冠 → 成本自适应可行。

**规律 C（鸿沟 × 增益）**：查询↔技能语义鸿沟越大，想象增益越大——medcalc（dense 0.139→满分）、logicbench（0.069→0.439）最悬殊；champ/bigcode 字面信号强的域增益收窄。

## 3. 与已发表方法的对比

- 超过 SRA 原装全部基线（含其论文报告的最强者 LLM 重排——同一个 Qwen3.5-4B，用来"想象"优于用来"重排"，且无 BM25 召回天花板锁死问题）；
- hybrid/llm_rerank 在 3 个域的补跑单元格待全量阶段补齐（rerank 会话中断）；SkillRouter（需下载 1.2B 模型）等外部方法列入全量阶段对比清单。

## 4. Go/No-Go 判定：GO

五域全冠、机制规律清晰、成本可控（试点全程本地模型零 API 费）。下一步：
